# Geprek Jago 🍗

Project sistem informasi untuk pengelolaan usaha Geprek Jago.

## Fitur Utama
* Manajemen Menu
* Catatan Transaksi Penjualan
* Laporan Harian

## Cara Instalasi
1. Clone repositori ini.
2. Pindahkan folder ke `C:\laragon\www\`.
3. Import database `geprek_jago.sql` melalui phpMyAdmin.
4. Jalankan server Laragon.

## Tech Stack
* PHP (Native/Laravel)
* MySQL
* Bootstrap
